**Первая задача**

Написать любой скрипт, который будет выполняться 5 секунд. Вшить в этот скрипт защиту от повторного запуска, если он еще не отработал. Использовать Redis. Не использовать фреймворки, написать на чистом php.

***Required***

- Redis
- Composer
- Php 8^

```
cd werenttest/task1
composer install
php src/script.php
```
